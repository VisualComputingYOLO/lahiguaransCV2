PROCESSING EXAMPLES - P5.JS EXAMPLES

Los siguientes ejemplos tienen el propósito de desarrollar y practicar la transición entre el lenguaje Processing y el lenguaje p5.js observando sus principales diferencias, ya que ambos lenguajes se ven muy similar, y cómo convertir de uno a otro.

Este programa esta basado en un sketch básico realizado por el sitio web mywonderland.es , en el lenguaje Processing, con el fin de observar un ejemplo básico donde el mouse interactúe con el lienzo. Crea una ventana con un ancho de 600px y una altura de 400px, y luego empieza a dibujar elipses de colores aleatorios en la posición donde se encuentre el mouse al mismo tiempo se ejecuta una función que calcula la velocidad con la que el mouse se mueve de una posición inicial a una posición final y según este valor determinar el tamaño de la elipse.

p5*js Code

																				
function setup() { 
	var myCanvas = createCanvas(windowWidth-300, 400);
	myCanvas.parent('speedMouse');
	background(210);
} 

function draw() {
	variableEllipse(mouseX, mouseY, pmouseX, pmouseY);
}

// El método simple variableEllipse () fue creado específicamente
// para este programa. Calcula la velocidad del mouse
// y dibuja una pequeña elipse si el mouse se mueve lentamente
// y dibuja una elipse grande si el mouse se mueve rápidamente

function variableEllipse(x, y, px, py) {
	let speed = abs(x - px) + abs(y - py);
	stroke(speed);
	fill(random(255),random(255),random(255));
	ellipse(x, y, speed, speed);
}

function windowResized() {
	resizeCanvas(windowWidth-300, 400);
	background(210);
}
